# Cloud chess AI
My New Project
Project Overview
Cloud Chess AI is a web-based chess platform built using Microsoft Azure services.
It allows users to:
- Play chess against a simple AI opponent.
- Store and review game history in a database.
- Compete on leaderboards with other players.
- (Optional) Learn from an AI coach that explains moves.
eatures
- Interactive chessboard with move validation.
- AI opponent powered by minimax algorithm.
- Game history stored in Azure SQL Database.
- Real-time multiplayer support with Azure SignalR.
- Deployed on Azure Web App for live access.
